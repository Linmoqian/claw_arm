# arm-control

## 描述

机械臂控制技能，用于控制 Galaxea R1 Lite 双臂机器人执行物理操作。
支持两种控制模式：
1. **SDK 直连模式** —— 通过飞特（FeeTech）舵机 UART 串口直接控制关节
2. **ROS2 模式** —— 通过 ROS2 话题/服务接口控制机械臂

机器人配置：
- 机型：galaxea_r1_lite
- 双臂结构：左臂 6 关节 + 夹爪，右臂 6 关节 + 夹爪
- 舵机型号：飞特 ST-3215-C001 (7.4V)，齿轮比 1:345
- 通信方式：UART 串口

## 使用场景

- 当用户要求机械臂执行抓取、放置、倒水等物理任务时
- 当需要控制机械臂单个关节或多关节协调运动时
- 当需要打开或关闭夹爪时
- 当需要机械臂回到初始位置时
- 当需要执行由 Lerobot 策略模型生成的动作轨迹时
- 当需要读取机械臂当前关节状态时

## 使用指令

```
帮我控制机械臂回到初始位置
帮我打开左臂夹爪
帮我关闭右臂夹爪
帮我读取机械臂当前状态
帮我执行倒水动作
帮我将左臂第3关节转到 1.5 弧度
```

## 权限要求

- 串口访问权限（/dev/ttyUSB0 或 COM 端口）
- ROS2 环境（如使用 ROS2 模式）
- Python 3.8+

## 依赖

- pyserial（串口通信）
- numpy（数据处理）
- rclpy（ROS2 模式，可选）

## 参数说明

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| mode | str | "sdk" | 控制模式："sdk" 或 "ros2" |
| port | str | "/dev/ttyUSB0" | 串口端口号 |
| baudrate | int | 1000000 | 串口波特率 |
| arm | str | "left" | 操作手臂："left"、"right" 或 "both" |
