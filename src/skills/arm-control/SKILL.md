# arm-control

## 描述

SO100 机械臂控制技能，通过 LeRobot 框架的 `SO100Follower` API 控制 6-DOF 机械臂。

机器人配置：
- 机型：SO100（6 自由度单臂）
- 电机：6 个飞特舵机（ID 1-6）
- 通信：USB 串口（默认 COM7）
- 电源：12V DC 适配器
- 控制框架：LeRobot (`lerobot.robots.so_follower`)

关节列表：

| 电机 ID | 关节名称 | 描述 | 位置范围 |
|---------|----------|------|----------|
| 1 | shoulder_pan | 底部旋转 | -100 ~ 100 |
| 2 | shoulder_lift | 大臂升降 | -100 ~ 100 |
| 3 | elbow_flex | 小臂弯曲 | -100 ~ 100 |
| 4 | wrist_flex | 手腕俯仰 | -100 ~ 100 |
| 5 | wrist_roll | 手腕旋转 | -100 ~ 100 |
| 6 | gripper | 夹爪开合 | 0 ~ 100 |

## 使用场景

- 当用户要求机械臂执行抓取、放置、倒水等物理任务时
- 当需要控制机械臂单个关节或多关节运动时
- 当需要打开或关闭夹爪时
- 当需要机械臂回到初始位置（零位）时
- 当需要读取机械臂当前各关节位置时
- 当需要执行预定义的动作序列时
- 当需要执行由 Lerobot 策略模型生成的动作轨迹时

## 使用指令

```
帮我读取机械臂当前状态
帮我控制机械臂回到初始位置
帮我打开夹爪
帮我关闭夹爪
帮我让 shoulder_pan 关节转到 30
帮我把 elbow_flex 设为 -45
帮我执行倒水动作
帮我执行抓取动作
帮我查看所有关节信息
```

## 权限要求

- USB 串口访问权限（COM7 或 /dev/ttyUSB0）
- Python 3.10+（推荐）
- LeRobot 框架已安装

## 依赖

- lerobot[feetech]（LeRobot + 飞特舵机支持）
- numpy（数据处理）
- pyserial（串口通信，随 lerobot 安装）

## 参数说明

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| port | str | "COM7" | USB 串口端口号 |
| robot_id | str | "my_so100_arm" | 机械臂唯一标识符 |
| calibrate | bool | false | 连接时是否执行校准 |
| max_relative_target | float | None | 单次最大相对移动幅度（安全限制） |

## 重要提示

1. **首次使用**必须先运行 `lerobot-setup-motors` 设置电机 ID
2. **首次使用或更换电机后**必须运行 `lerobot-calibrate` 进行校准
3. 关节位置值范围为 **-100 到 100**（夹爪为 **0 到 100**），不是弧度
4. 超出范围的值会被自动裁剪到安全范围
5. 断开连接时会自动禁用所有电机扭矩
