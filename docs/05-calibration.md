# 相机标定

本文档介绍如何进行相机标定以提高坐标转换精度。

## 为什么需要标定？

相机镜头存在畸变，像素与真实距离不是简单的线性关系。标定可以：
- 消除镜头畸变
- 获取相机内参（焦距、光心）
- 提高坐标转换精度

## 标定原理

### 相机内参矩阵

```
    ┌ fx  0  cx ┐
K = │ 0  fy  cy │
    └ 0   0   1 ┘

fx, fy = 焦距（像素）
cx, cy = 光心（像素）
```

### 畸变系数

```
D = [k1, k2, p1, p2, k3]

k1, k2, k3 = 径向畸变
p1, p2 = 切向畸变
```

## 标定工具

### 方法一：使用 OpenCV

项目提供了标定脚本：

```bash
# 1. 采集标定图像
python script/2.1capture.py

# 2. 执行标定
python script/2.2calibrate_camera.py
```

### 方法二：使用棋盘格

1. 准备棋盘格标定板（推荐 9x6 内角点）
2. 打印并固定在平板上
3. 从不同角度拍摄 15-20 张图像
4. 运行标定程序

## 标定流程

### Step 1: 采集图像

```bash
python script/2.1capture.py
```

操作说明：
- 从不同角度拍摄标定板
- 确保标定板完整出现在画面中
- 覆盖画面的不同区域

### Step 2: 执行标定

```bash
python script/2.2calibrate_camera.py
```

输出示例：
```
检测到 20 张有效图像
重投影误差: 0.23 像素

相机内参:
[[800.5,   0.0, 960.2],
 [  0.0, 801.3, 540.1],
 [  0.0,   0.0,   1.0]]

畸变系数:
[-0.02, 0.01, 0.0, 0.0, 0.0]
```

### Step 3: 验证结果

```bash
python CV/4_world_coordinate.py
```

检查转换后的坐标是否准确。

## 手眼标定

手眼标定确定相机与机械臂之间的位置关系。

### Eye-to-Hand 配置

相机固定，机械臂移动：

```
标定流程:
1. 控制机械臂移动到多个位置
2. 在每个位置记录机械臂位姿和标定板位置
3. 求解变换矩阵
```

### 执行手眼标定

```bash
python script/2.4hand_eye_calibration.py
```

## 标定结果存储

标定结果保存在 `camera_config.json`：

```json
{
  "camera_matrix": [
    [800.5, 0.0, 960.2],
    [0.0, 801.3, 540.1],
    [0.0, 0.0, 1.0]
  ],
  "dist_coeffs": [-0.02, 0.01, 0.0, 0.0, 0.0],
  "fixed_height": 0.3
}
```

## 标定质量评估

| 指标 | 优秀 | 良好 | 需要重新标定 |
|------|------|------|--------------|
| 重投影误差 | < 0.2 px | < 0.5 px | > 1.0 px |
| 畸变系数 | < 0.1 | < 0.3 | > 0.5 |

## 常见问题

### 标定失败？

1. 确保标定板平整
2. 增加采集图像数量
3. 检查光照条件

### 精度不够？

1. 使用更大尺寸的标定板
2. 采集更多角度的图像
3. 确保标定板完全清晰

### 结果不稳定？

1. 固定相机位置
2. 避免震动
3. 使用稳定的光源
