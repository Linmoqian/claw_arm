## 硬件参数

- **机型**: SO100 (6-DOF 单臂)
- **串口**: COM7
- **波特率**: 1000000
- **电机数量**: 6 (ID 1-6)
- **位置范围**: 0 ~ 4095（中位 2047）

## 使用方法

```bash
# Windows (必须设置编码)
PYTHONIOENCODING=utf-8 python skills-openclaw/nl-arm-control/scripts/executor.py

# 直接执行指令
PYTHONIOENCODING=utf-8 python skills-openclaw/nl-arm-control/scripts/executor.py "张开夹爪"
```
